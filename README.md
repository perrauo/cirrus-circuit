# Cirrus Circuit

* Multiplayer game where each opponent push gems into the door of the same color
* Chain multiple entries for combos
* Push the wrong color and lose points.

# Unity version
2019.3.0f6

# Git commands
* Pull repo and submodules
    * `./cirrus-pull.sh master`
* Update submodules
    * `./subm-pull.sh`
    * `./subm-add.sh`
    * `./subm-commit.sh`
    * `./subm-push.sh`
